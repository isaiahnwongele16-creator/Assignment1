The Defferent B/W This Following

  • ✅Viewport
  • ✅Mega Tag
  • ✅Media Queries
  • ✅Transition
  • ✅Transform
  • 1. Viewport: The visible area of a web page on a device. Makes websites display correctly on phones, tablets, and computers.
  • 2. Mega Tag: (not "Mega tag") An HTML tag placed inside the section. Provides information about the page, such as character encoding, description, and viewport settings.
  • 3. Media Queries: A CSS feature. Applies different styles based on screen size, orientation, or device type to create responsive designs.
  • 4. Transition: A CSS property. Creates a smooth animation when a CSS property changes, such as on hover.
  • 5.Transform: A CSS property. Changes the appearance of an element by moving, rotating, scaling, or skewing it.
